This Stone Paper Scissors project is a simple project coded in HTML(HyperText Markup Language), CSS(Cascading Style Sheet) and JavaScript.
We can pick the player move by clicking each icon given.Additional Features like Auto-play and KeyBoard playing are enabled.

Instructions:
      => Functions of key(case insensitive):
           r-Rock
           p-Paper
           s-Scissors
      => Click Auto play button and wait for 2 seconds to make the computer play with itself.
      => Click reset button to reset the scores as the previous game will always get counted even if you refresh the page.
      => Click auto play to stop playing.
